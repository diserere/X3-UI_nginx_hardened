# 3X-UI + Nginx Reality Infrastructure (Hardened)

Проект по развертыванию защищенной, портируемой инфраструктуры реверс-прокси для панели 3X-UI и трафика VLESS+Reality с разделением потоков по SNI на сетевом уровне ядра.

## 🚀 Архитектурная концепция
- **Frontend:** Nginx (порт 443) распределяет трафик на уровне TCP (модуль `stream`) на основе SNI без терминирования TLS для Reality.
- **Backend:** 3X-UI (панель, подписки) + Xray (Reality TCP) внутри изолированной Docker-сети.
- **DNS:** DuckDNS (поддомены 4-го уровня для разделения сервисов панели и подписок).
- **Парадигма:** Configuration-as-Code / Context-as-Code (полный Docker-стэк, управление через ИИ-агентов).

## 🛠 Быстрый старт (Развертывание)
1. Скопировать `src/certbot/.env.example` в корень под именем `.env` и заполнить переменные (токен DuckDNS, email, субдомен).
2. Запустить стек:
   ```bash
   docker compose up -d
   ```
