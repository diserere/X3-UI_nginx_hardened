## 🛠️ Правила работы с бэклогом для ИИ-агентов
1. **Взятие в работу**: При начале работы над задачей агент обязан изменить статус задачи с `Open` на `In Progress` и зафиксировать это в `docs/05_interconnection_log.md`.
2. **Завершение**: После успешного тестирования и слияния кода в `main`, статус задачи меняется на `Closed`.
3. **Добавление новых**: Новые инженерные задачи добавляются строго в конец таблицы с инкрементом ID.
